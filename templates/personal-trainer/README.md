# Template Personal Trainer Premium (Divinópolis)

Este template foi criado para personal trainers em Divinópolis e pode ser customizado rapidamente.

## Como personalizar (em 5 minutos)

### 1) Nome e título
No arquivo `index.html`, troque:
- **Carolina Silva** pelo seu nome.
- **Personal Trainer Premium em Divinópolis** pelo seu título.

### 2) WhatsApp
No arquivo `index.html`, procure por `wa.me/5537999999999` e substitua pelo seu número.

Exemplo:
```
https://wa.me/5537999999999
```

Troque para:
```
https://wa.me/5537SEUNUMERO
```

### 3) CREF e credenciais
No arquivo `index.html`, ajuste:
- `CREF 000000-G/MG`
- `10+ anos`
- `500+ alunos`

### 4) Resultados e depoimentos
No arquivo `index.html`, edite os blocos:
- **Resultados em destaque** (métricas)
- **Depoimentos** (textos e nomes)

### 5) Cidade
Se não for Divinópolis, atualize:
- `Divinópolis` no texto e metadados.
- `addressLocality` e `addressRegion` no JSON-LD.

## Arquivos principais
- `index.html` → conteúdo do site.
- `style.css` → estilos visuais.
- `script.js` → interações (menu, FAQ, copiar email, scroll).
